Zynga Scroller

Accelerated panning and zooming for DOM and Canvas

Dependencies

Zynga Scroller has no dependencies to other JavaScript libraries.

Demo

  • DOM Demo
  • Canvas Demo
  • Paging X-Axis
  • Paging Y-Axis
  • Snapping to Cells

Install

  • Script tag use: this is the use case supported by Zynga's fork. Include "animate.js" followed by "Scroller.js".
  • CommonJS: npm install scroller.
  • AMD: bower install popham/scroller OR npm install scroller.
